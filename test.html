Instructions

  1. Install the AI Block extension in Chrome (see README.md)
  2. Open this test page in Chrome
  3. Click the extension icon to open settings
  4. Enable/disable blocking and see the results
  5. Elements with red borders (ads) should be hidden when blocking is enabled

How It Works

  • Content Script: Runs on every page and applies blocking rules
  • CSS Selectors: Uses domain-specific selectors to find elements
  • Dynamic Blocking: MutationObserver watches for new elements
  • Storage: Settings persist across browser sessions

Default Supported Sites

  • Amazon: Sponsored products and ads
  • Google: Search ads and sponsored results
  • YouTube: Display ads and promoted content
  • Facebook: Sponsored posts
  • Twitter/X: Promoted tweets
  • Reddit: Promoted posts
